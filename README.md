# 审问Notice

### 教务处通知 `jwnotice`
> "platform": "jw"

#### 参数
| 参数名     | 含义              | 默认值 |
|---------|-----------------|----|
| name    | 通知类型(`jw`/`xy`) | jw |
| tag     | 通知标签            | 0  |
| pageNum | 页码              | 1  |
| pageSize | 分页大小            | 15 |



#### 教务通知`jw`

| tag | 含义     |
|-----|--------|
| 0   | 全部教务通知 |
| 1   | 选课     |
| 2   | 考试     |
| 3   | 实践     |
| 4   | 交流     |
| 5   | 教师     |
| 6   | 信息     |

#### 学院通知`xy`

| tag | 含义     |
|-----|--------|
| 0   | 全部学院通知 |
| 1   | 选课     |
| 2   | 考试     |
| 6   | 信息     |
